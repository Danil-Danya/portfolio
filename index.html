Умения›

HTML

Html незаменимый инструмент для веб разработчика. У меня, как и у многих веб разработчиков начался путь с него. Умение писать код по Бэму, делает его чистым и понятным. (Есть опыт работы с прицоцессором pug).

CSS

Уметь верстать - главная обязанность любого фронтенд разработчика. Гриды, флексы, адаптивная верстка, верстка psd макетов, макетов в фигме, все уже нераз делал.

BOOTSTRAP

Людям не знающим языков программирования будет удобно использовать фреймворк bootstrap. Удобыный фреймворк где за тебя уже все сделанно и думать ни очем кроме верстки не надо. Удобная сетка позволяет быстро верстать адаптивку, а уже сделанные веб приложения намного упрощают работу. У него так же есть одно но: большенство js компонентов неанимированны.

JAVASCRIPT

Незаменимый язык программирования во фронтенд разарботки. Знание этого языка позволяет создовать кастомные веб приложения. Знание стандарта языка ecmaScript 6 позволяет писать в обновленном синтаксисе.

JQUERY

Удобная библиотека по JavaScript'у. Дает возможность быстро подключиться к DOM елементам. Предамтвляет из себя набор функций, методов, которые можно удобно связыать в цупочку. У этой библиотеки есть большой минус, что она сильно нагружает сайт.

VUE

Vue.js - удобный, мощный, быстрый фреймворк. В последнее время сильно закрепился на рынке, обошел angular и конкурирует с react. Его вируальный DOM помогает оптимизировать сайт, а интреполяция позволяет взаимодейтсвовать с JavaScript'ом в html.

GULP

Gulp мощный сборщик, который подходит для новичков. Его сборка может подкючить необходимые компоненты для оптимизации сайта. Позволяет разбивать приложения на компоненты для структуризации кода.

SASS

Scss, sass мощный прироцессор для css. Он делает код более структуированным и хорошо читабельным. Его главная прелесть в том что код можно писать только по бэму (иначе он будет бесполезным).

PHP

Php серверный язык программирования. Учил его недолго для пониамания бекенда. Его низкий порог входа помогает быстро ориентироваться в нем новичкам, учится очень лекго после JavaScript.
